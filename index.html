GitHub Gist Sync

How to get your GitHub Token (2 min):
1. Go to github.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Give it a name (e.g. "Bijli Tracker"), check only the gist scope
4. Click Generate → Copy the token below
5. Leave Gist ID blank on first save — app will create a private Gist automatically ✅

🔒 Token is stored only in your browser's localStorage — never sent anywhere except GitHub's API.
